---
title: 什麼是 Hollo？
description: Hollo 是一款面向單用戶場景的聯邦式微博軟體。
---

Hollo 是一個簡單的單用戶微博工具，讓你可以在網路上擁有自己的小天地。你可以將它視為你的個人 Mastodon 實例，但僅保留了核心功能。

Hollo 的獨特之處在於它是 [聯邦宇宙](https://www.theverge.com/24063290/fediverse-explained-activitypub-social-media-open-protocol) 的一部分——一個運行在開放協定上的互聯伺服器網路，主要使用 [ActivityPub](https://activitypub.rocks/)。這意味著你可以與其他平台上的用戶連接和互動，比如：

- [Mastodon](https://joinmastodon.org/zh)
- [Misskey](https://misskey-hub.net/)
- [Akkoma](https://akkoma.social/)
- [WordPress](https://wordpress.org/)
- [Threads](https://www.threads.net/)
- 以及許多其他聯邦宇宙伺服器

這樣，你既能享受個人空間的寧靜，又能連接到更廣泛的社群。

功能
----

Hollo 在大多數單用戶需求上與 Mastodon 功能相當。你可以獲得所有核心微博功能，但不會有不適合單用戶實例的部分（如管理工具）。這意味著你可以：

- 使用標籤和提及
- 分享媒體附件
- 建立投票
- 轉發和收藏貼文
- Misskey 風格的表情回應

在撰寫貼文時，Hollo 保持靈活性。你可以：

- 撰寫最多 10,000 字的貼文
- 使用 [CommonMark](https://commonmark.org/)（即 Markdown）進行輕鬆格式化
- 使用符合 [FEP-044f](https://w3id.org/fep/044f) 標準的引用（Misskey 風格和 Mastodon 風格）為你的貼文增添風采

Hollo 是所謂的「無頭」軟體，意思是它沒有自己的網頁介面。相反，你可以使用任何與 Mastodon 相容的應用程式進行發布和互動。乍一聽可能有些奇怪，但實際上這讓你可以選擇最適合你風格的應用。

本質上，Hollo 是一個簡單的工具，將 Mastodon 或 Misskey 的力量帶給個人用戶。它幫助你在線上分享想法——從快速更新到更長的思考——在聯邦宇宙中與他人連接，並以自己的方式進行。它並不想成為下一個大型社群網路——只是希望讓微博變得更簡單、更連通。

詞源
----

名稱 *Hollo* 源自韓語詞 *홀로*，意為「獨自」或「孤身」。之所以這樣命名，是因為它被設計為單用戶軟體。
