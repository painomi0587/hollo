Hollo 可以手動安裝在您的伺服器上。本指南將引導您完成在伺服器上設定 Hollo 的過程。假設您有執行 Web 應用程式的經驗，並且熟悉命令列操作。

<Aside type="tip">
  如果您對類 Unix 系統不熟悉或更偏好自動化的方式部署 Hollo，可以使用 [Railway] 僅需幾次點擊即可部署 Hollo。

  [Railway]: /zh-tw/install/railway
</Aside>


準備工作
--------

開始之前，請確保您的伺服器上已安裝以下軟體：

 -  [Git]
 -  [Node.js] 24+
 -  [pnpm]
 -  [ffmpeg]
 -  [PostgreSQL] 17+
 -  L7 負載均衡器（例如，[nginx]，[Caddy]）
 -  指向您伺服器的網域名稱

[Git]: https://git-scm.com/
[Node.js]: https://nodejs.org/
[pnpm]: https://pnpm.io/
[ffmpeg]: https://www.ffmpeg.org/
[PostgreSQL]: https://www.postgresql.org/
[nginx]: https://nginx.org/
[Caddy]: https://caddyserver.com/


安裝
----

import { Steps } from "@astrojs/starlight/components";

<Steps>
 1. 從 [GitHub] 取得 Hollo 的最新程式碼：

    ~~~~ sh frame="none"
    git clone -b stable https://github.com/fedify-dev/hollo.git
    cd hollo/
    ~~~~

 2. 使用 pnpm 安裝相依套件：

    ~~~~ sh frame="none"
    pnpm install
    ~~~~

 3. 為 Hollo 建立 PostgreSQL 使用者和資料庫：

    ~~~~ sh frame="none"
    createuser --createdb --pwprompt hollo
    createdb --username=hollo --encoding=utf8 --template=postgres hollo
    ~~~~

 4. 為 Hollo 建立設定檔：

    ~~~~ sh frame="none"
    cp .env.sample .env
    ~~~~

[GitHub]: https://github.com/fedify-dev/hollo
</Steps>


設定
----

安裝 Hollo 後，您需要進行設定。開啟之前建立的 *.env* 檔案並調整環境變數。

有關如何設定 Hollo 的詳細資訊，請參閱 [**環境變數**](/zh-tw/install/env) 章節。


啟動伺服器
----------

先建構 JavaScript 檔案，然後啟動伺服器：

~~~~ sh frame="none"
pnpm run build
pnpm run prod
~~~~


升級 Hollo
----------

要升級 Hollo，只需從 GitHub 拉取最新程式碼並重新安裝相依套件：

<Steps>
 1. 從 GitHub 拉取最新程式碼：

    ~~~~ sh frame="none"
    git pull
    ~~~~

 2. 重新安裝相依套件：

    ~~~~ sh frame="none"
    pnpm install
    ~~~~

 3. 重新建構 JavaScript 檔案：

    ~~~~ sh frame="none"
    pnpm run build
    ~~~~

 4. 重新啟動伺服器：

    ~~~~ sh frame="none"
    pnpm run prod
    ~~~~
</Steps>
